# Security Policy

## Supported Versions

Security fixes are provided for the latest published minor line of `@potetotown/vitrio`.

## Reporting a Vulnerability

Please report vulnerabilities privately by opening a **private security advisory** on GitHub (preferred).
If that is not available, open an issue marked `security` without exploit details and maintainers will follow up privately.

Include:
- affected version
- reproduction steps
- impact assessment
- any known mitigations

## Response Targets

- Initial acknowledgement: within 3 business days
- Triage update: within 7 business days
- Fix/release target: depends on severity and complexity

## Disclosure

We follow coordinated disclosure. Please do not publish details before a fix or mitigation is available.
