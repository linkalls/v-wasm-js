# Support Policy

## Runtime Support

- Node.js: active LTS versions
- Bun: latest stable version
- Browsers: modern evergreen browsers (Chrome/Edge/Firefox/Safari)

## Versioning and Maintenance

- Semantic Versioning (`MAJOR.MINOR.PATCH`)
- Breaking changes are released only in major versions
- Critical bug/security fixes are prioritized for the latest minor line

## What "Supported" Means

Supported versions receive:
- bug fix review
- security triage
- documentation corrections

Unsupported versions may still work but are not guaranteed to receive fixes.
